Java Compiler Version 1.5 required

To use this jar you need the following libs which can be found at:

* Codec http://jakarta.apache.org/commons/codec/
** commons-codec-1.3.jar

* HttpClient http://jakarta.apache.org/commons/httpclient/
** commons-httpclient-3.0.1.jar

* Logging http://jakarta.apache.org/commons/logging/
** commons-logging-1.1.jar
** commons-logging-adapters-1.1.jar
** commons-logging-api-1.1.jar

* Log4J http://logging.apache.org/
** log4j-1.2.14.jar

* HtmlParser http://htmlparser.sourceforge.net/
** htmlparser.jar

* Junit http://www.junit.org/
** junit-4.3.1.jar


or get it from the SVN
